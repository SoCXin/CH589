[OM6621EM](https://www.onmicro.com.cn/portal/article/index/cid/40/id/782.html)  

### 关键特性

* 64 MHz Cortex-M4
* 40KB SRAM + 512KB Flash
* 256-bit EFUSE
* 12bit ADC 500Kbps
* 16bit Audio ADC (93dB)
* 1x PMIC + 1x AMIC
* BLE 5.1
    * RX -97dBm @ 1Mbps
    * TX 7dBm
* sleep < 2.96uA
* BLE RX < 4.9mA
* BLE TX 4.95mA @ 0dBm
* QFN32 (23x GPIO)
